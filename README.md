# Шаблон для лабораторных и курсовых работ (TeX)

Этот проект представляет собой шаблон для оформления лабораторных работ и курсовых проектов в системе LaTeX для СПбГУАП.

## Структура проекта

### Основные файлы
*   `preamble.tex` — файл с преамбулой. Здесь подключены все необходимые пакеты и настроено оформление.
*   `laboratory.tex` — титульный лист для лабораторной работы.
*   `coursework.tex` — титульный лист для курсовой работы.

## Как пользоваться

### Для лабораторной работы

Чтобы изменить данные на титульном листе (имя студента, преподавателя, название работы и т.д.), откройте файл `laboratory.tex` или `coursework.tex` и отредактируйте соответствующие команды в начале файла:

```latex
% Данные преподавателя
\newcommand{\Dep}{99}                               % Кафедра
\newcommand{\teacherPos}{профессор, д-р техн.наук}  % Должность
\newcommand{\teacherName}{Петров П.П.}              % Имя преподавателя

% Данные лабораторной работы
\newcommand{\labNum}{66} % Номер
\newcommand{\labTitle}{Название вашей лабораторной работы}  % Название
\newcommand{\courseName}{Название дисцпилины}               % Название дисциплины

% Данные студента
\newcommand{\studentGroup}{м1123пс}          % Номер группы
\newcommand{\studentName}{Иванов И.И.}       % Имя студента
\newcommand{\cityYear}{2026}                 % Год

```

Затем, скопируйте файл в директорию своего проекта (в файле компиляции используйте `\input{}`) или вставьте содержимое прямо в ваш файл компиляции.

### Для курсовой работы

Для курсовой работы откройте файл `coursework_content.tex` и отредактируйте параметры проекта:

```latex
% Данные преподавателя
\newcommand{\Dep}{ИНФОКОММУНИКАЦИОННЫХ ТЕХНОЛОГИЙ И СИСТЕМ СВЯЗИ} % Кафедра
\newcommand{\teacherPos}{доцент, канд. техн. наук}                % Должность
\newcommand{\teacherName}{В.Н. Ершов}                             % Имя преподавателя

% Данные лабораторной работы
\newcommand{\labTitle}{РАЗРАБОТКА ЭМУЛЯТОРА ШИФРОВАЛЬНОЙ МАШИНЫ «ЭНИГМА» НА БАЗЕ ESP8266} % Название
\newcommand{\courseName}{программно-аппаратные средства защиты}                           % Название дисциплины

% Данные студента
\newcommand{\studentGroup}{м1123пс}          % Номер группы
\newcommand{\studentName}{Иванов И.И.}       % Имя студента
\newcommand{\cityYear}{Санкт-Петербург 2026} % Город и год
```

Используется аналогично `laboratory.tex`.

### Использование преамбулы

Файл `preamble.tex` лучше всего подключать с помощью `\input{}`, а не копировать содержимое  это файла в свой проект (из-за объемного содержимого).

В преамбуле настроены:
*   Поля и отступы по ГОСТ.
*   Шрифты (Times New Roman).
*   Оформление заголовков, оглавления, рисунков и таблиц.
*   Библиография (biber).
*   Подсветка кода (minted).
